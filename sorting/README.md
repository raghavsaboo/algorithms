# Overview of Sorting Algorithms

|                  | Insertion Sort | Selection Sort |  Merge Sort  |  Quick Sort  |
| :--------------: | :------------: | :------------: | :----------: | :----------: |
|  **Best Case**   |     `Ω(n)`     |    `Ω(n^2)`    | `Ω(nlog(n))` | `Ω(nlog(n))` |
| **Average Case** |    `θ(n^2)`    |    `θ(n^2)`    | `θ(nlog(n))` | `θ(nlog(n))` |
|  **Worst Case**  |    `O(n^2)`    |    `O(n^2)`    | `O(nlog(n))` |   `O(n^2)`   |
